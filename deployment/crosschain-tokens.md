# CIOTX (Crosschain IOTX)

Crosschain IOTX or `CIOTX`, is a _wrapped_ token of IOTX that can be bridged to any network via [iotube.org](https://iotube.org). You can wrap or unwrap a token at a 1:1 ratio anytime.&#x20;
